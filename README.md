### Installation

You need to have Node installed and then run this command in the project folder:

```
npm install
```
or:
```
yarn install
```
Depending on your package manager.

Then you run the command: 
```
node src/index.js
```
or: 
```
yarn dev
```
To run the developer dependency Nodemon.
